This Project is the Frontend part of the QuranSchool Project.

https://github.com/haiderali711/QuranSchool

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
